## How to build?

1. Clone repo

```
git clone https://github.com/dmitrytavern/object-parser.git
```

2. Install deps

```
npm install
```

3. Build

```
npm run build
```

Other commands:

```
npm run dev - start hot reload server
npm run build:serve - run express server for dist
```
